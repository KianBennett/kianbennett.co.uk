Ank'hur
View on itch.io
Ank'hur is a puzzle game made from a board of sliding blocks. The goal of the game is to get each basic block into a target square - the player can tilt the board one of four directions and different blocks will slide in different ways. The game has five sets of five levels, with each set introducing a new block type.

Players can create their own levels and upload them for other players to play and rate!
