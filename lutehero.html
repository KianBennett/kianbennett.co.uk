Lute Hero
View on itch.io
Lute Hero is an rpg/rhythm game with a focus on Medieval and Renaissance music, combining the two genres of rhythm and turn-based role playing. It is built in Unity with C# - I made all the art but the song recordings are from Jon Sayles. Some notable features of the game are:
  • Detailed rhythm input system for playing notes of a song
  • Turn-based combat
  • Inventory with equippable and consumable items
  • Multiple towns, each with their set of townsfolk with individual daily routines
  • Song chart parser allowing for custom songs to be loaded into the game (cross compatible with custom Guitar Hero charts)
